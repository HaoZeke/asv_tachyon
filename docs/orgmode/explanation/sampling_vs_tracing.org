#+TITLE: Sampling metrics vs the results UI
#+OPTIONS: toc:nil

*asv-tachyon* is the results UI. *asv_bench_tachyon* is the =sample_*= metric
plugin (Python 3.15 =profiling.sampling=). *asv_bench_memray* is the =ray_*=
peak-memory plugin.

Both metric plugins can write HTML under =.asv/profiles/<kind>/= and publish
into =profiles.json= so Explore can open them. The UI never runs a sampler.
