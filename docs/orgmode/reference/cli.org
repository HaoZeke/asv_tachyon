#+TITLE: CLI
#+OPTIONS: toc:nil

=asv-tachyon= exposes two subcommands:

| Command | Purpose |
|---------+---------|
| =asv-tachyon serve [html_dir]= | Serve modern UI over published results |
| =asv-tachyon install html_dir= | Copy SPA shell into =html_dir= |

There is no =sample= / =replay= CLI here. Sampling metrics and flamegraph
artifacts come from =asv_bench_tachyon= / =asv_bench_memray=.
