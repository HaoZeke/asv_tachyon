* Output formats

| Format | Flag / =--format= | Notes |
|--------+-------------------+-------|
| pstats | =pstats= | Text or binary pstats |
| flamegraph | =flamegraph= | Interactive HTML |
| heatmap | =heatmap= | Per-file line heatmaps (directory) |
| gecko | =gecko= | Firefox Profiler JSON |
| collapsed | =collapsed= | Folded stacks |
| binary | =binary= | Compact capture for =replay= |
| diff-flamegraph | =diff-flamegraph= | Needs =--baseline= |
