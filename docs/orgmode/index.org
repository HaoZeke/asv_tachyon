#+begin_export rst
asv-tachyon
===========

:Author: `Rohit Goswami <https://rgoswami.me>`_
#+end_export

Modern web UI for airspeed velocity results. Drop-in shell over
=asv publish= data. Screenshots and Sphinx pages live under =docs/source/=
(primary docs entrypoint) — open the built site or the GitHub README gallery.

* Install

#+begin_src bash
pip install asv-tachyon
asv publish
asv-tachyon serve .asv/html --open
#+end_src

Primary documentation is the Sphinx tree in =docs/source/= (Overview,
Compare, Inventory screenshots). Legacy sampling tutorials under this
orgmode tree are historical and not the product surface.
