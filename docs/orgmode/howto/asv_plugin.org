Optional HTML opener for =asv profile --gui=tachyon=:

#+begin_src json
{ "plugins": ["asv_tachyon.plugin"] }
#+end_src

Opens Tachyon / memray HTML reports. For cProfile dumps, use snakeviz.
For sampling metrics, install =asv_bench_tachyon= and enable
=sample_save_profile= (or =ASV_BENCH_TACHYON_PROFILE=1=).
