* asv-tachyon

Modern web UI for [[https://asv.readthedocs.io/][airspeed velocity]] results.
Drop-in shell over =asv publish= data. Profile *collection* lives in
=asv_bench_tachyon= / =asv_bench_memray=; this package only serves the site
and optional =profiles.json= links.

* Install

#+begin_src bash
pip install asv-tachyon
asv publish
asv-tachyon serve .asv/html --open
#+end_src

Demo: =asv-tachyon serve fixtures/sample_site --open=

* Profiles

#+begin_src bash
ASV_BENCH_TACHYON_PROFILE=1 asv run --bench sample_
python -m asv_bench_tachyon profiles publish --html-dir .asv/html
asv-tachyon serve .asv/html --open
#+end_src
